Problem: Two Sum 
Platform: Leetcode
Difficulty: Easy
Approach: 
Sort the given array and use 2 pointers to traverse the array and find the elements whose sum is equal to the target.
Use a left pointer to traverse from beginning and a right pointer to travese from the end until both of them point to the same location
or the sum has been found.
Time Complexity:
O(n)
Space Complexity:
O(1)
